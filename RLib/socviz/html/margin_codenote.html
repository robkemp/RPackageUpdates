margin_codenote {socviz}R Documentation

margin_codenote

Description

Use in inline R expressions to write a monospaced margin note for tufte-latex documents

Usage

margin_codenote(text, icon = "&#8853;")

Arguments

text

The text of the note

icon

An icon

Details

Adapted from Dirk Eddelbuettel's tint package.

Value

A marginal note in monospace font

Author(s)

Kieran Healy

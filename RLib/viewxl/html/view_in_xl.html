view_in_xl {viewxl}R Documentation

Open a data.frame in 'Excel'

Description

Open a data.frame in 'Excel'

Usage

view_in_xl(df = NULL)

Arguments

df

The name of a data.frame or a data.frame itself, if not provided, an addin is launched.

Examples

## Not run: 

# Launch addin
view_in_xl()

# or pass the name of a data.frame
view_in_xl("iris")

# or directly the object
view_in_xl(iris)


## End(Not run)

arc.shape {arcgisbinding}R Documentation

Get arc.shape object

Description

Get arc.shape-class from arc.dataframe

Usage

arc.shape(df)

Arguments

df

arc.dataframe

See Also

arc.select

Examples


d <- arc.open(system.file("extdata", "ca_ozone_pts.shp",
                          package="arcgisbinding"))

df <- arc.select(d, 'ozone')
shp <- arc.shape(df)
length(shp$x)
